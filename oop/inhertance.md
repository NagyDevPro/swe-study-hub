# Inhertance

# Introduction
- Inhertance defined as **IS A** Relationship `Specialization` (Meaning each small class is a special type of another big type).
- the parent class can be called **Super Class**.
- the child class can be called **Sub/Derived Class**.
- Some Languages like `java` is build on oop, and have all classes and types are inherited from a base class called **Object**.
- A `final class` is a class that cannot be extented
